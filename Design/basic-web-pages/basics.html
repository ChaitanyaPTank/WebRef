Lists

This is how to make unordered list

  • Add a "ul" element (ul: unordered list)
  • Add each item in its own "li" element
  • They don't need to be in perticular order
  • You shoud not write any other tag in ul than the li. like <ul> <p>This is wrong</p></ul>
  • Inplace of this you should do: <ul><li><p>This is right</p></li></ul>

This is how ordered list look like:

  1. New "ol" element is wrapping everything
  2. But, the "li" is the same
  3. Also numbers will be get incremented on their own
  4. Things are in precise order because this is ordered list
